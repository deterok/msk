MSK GIT support mod
===================

Preparation
-----------

if you use gpg signature you may need to set environment variable "GNUPGHOME"

        (msk/setenv-if-not-exist "GNUPGHOME"
                                 (expand-file-name "~/PATH_TO_GNUPGHOME"))

Start
-----

Just run magit-start and press H for help on commands.
